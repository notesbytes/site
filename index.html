Advertisements
How we fund this
NotesBytes costs money to run, and we need to make it somehow. This is why we will be adding advertisements and how we earn a bit as well. We will add advertisements however we WILL KEEP IT AT A MINIMUM like Duolingo. We will also have paid plans which can include more helpful features!
Notes
The note exchange
We will be releasing and funding for our own notes in the future however currently users can post it for free or charge between £2 and £10,000, even better only 15% commission, lowest in comparison to similar platforms!

Payouts are done via PayPal only and are done in GBP, USD and EUR.
AI
Student Solutions
We also plan on adding AI to aid students and show them things in homeworks, using things like Wolfram Alpha and Tensorflow.js. We will of course be donating to Tensorflow and paying Wolfram Alpha for API Usage.
API
Developers using our system
We use a RESTful API to run the main NotesBytes backend. After releasing, we will allow developers to use our API with limits and add paid plans to have a better limit. We use limits since the API does cost us and if exploited, could result in a large charge for us.
